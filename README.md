# ShortLinker

ShortLinker est un petit service de **raccourcissement de liens** avec suivi des statistiques.  
Il permet de transformer un lien long en un lien court, et de suivre :  
- Le nombre de clics  
- Les dates de clic  
- L’adresse IP et le navigateur utilisé  

C’est un projet **basé sur Flask**, simple à utiliser et à personnaliser.

---

## **Fonctionnalités**

- Génération automatique de lien court
- Redirection vers le lien original
- Statistiques détaillées pour chaque lien
- Interface simple et épurée
- Possibilité d’extension avec graphiques et persistance
